fastlane插件

fastlane action是fastlane官方定义好的行为，比如:

  • scan 测试
  • gym 打包
  • cocoapods 运行pod install
  • clear_derived_data 清理Xcode的项目缓存数据
  • increment_build_number 自动增加build号
  • sigh 生成PP文件
  • 更多详见fastlane action docs

而fastlane的插件是允许开发者自己来创建action，满足开发者使用和定制自己需要的服务，比如fastlane的开发者就在自己的github里写了一个appicon插件，用于自动化制作出不同尺寸的app图标；还有支持上传蒲公英测试平台的pgyer插件，不用开启浏览器就直接上传ipa

这里官方文档中给出的一个插件列表

比如添加蒲公英插件，方法就是fastlane add_plugin pyger，fastlane会将安装插件并添加到Pluginfile中，完成后使用插件就跟使用action是一样的，可以在Fastfile中创建由插件提供方法的lane，关于怎么调用api和传递参数，在插件自己的

而安装appicon插件需要依赖于ImageMagick，如果没有安装的话，首先要brew install imagemagick

  • 理解 Scroll Views
  • Understanding UIScrollView

这样的话，为什么还需要contentOffset这个属性呢？因为调用setContentOffset:会触发滑动的行为，因此也会调用scrollViewDidScroll:的代理方法；而直接设置bounds.origin就不会。
